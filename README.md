# secure-personal-finance-manager
A secure, privacy-focused personal finance management system built with core PHP, PDO, and encrypted data storage.

## Features
- User authentication
- Encrypted financial data storage
- Multi-account management
- Budget tracking
- Financial reports and analytics

## Tech Stack
- PHP (Core)
- MySQL
- PDO
- Chart.js
- Bootstrap

## Security Measures
- Password hashing
- AES-256 encryption for sensitive fields
- Prepared statements (PDO)
- Session hardening

## Setup Instructions
1. Clone repository
2. Configure `.env`
3. Import database schema
4. Run on local server
